## ‚ú® Key Concepts Covered

### üõ†Ô∏è Notebook 01: Creation, Properties, and Basic Manipulation
This notebook focuses on setting up and understanding the structure of a Pandas DataFrame:

* **DataFrame Creation:** How to make pandas DataFrames from other Python collections (e.g., lists of lists, dictionaries).
* **Inspection:** Learning about your DataFrame's properties and information (e.g., `.shape`, `.dtypes`, `.info()`).
* **Column Operations:** Selecting a single column.
* **Modification:** Adding new columns to a DataFrame.
* **Renaming:** Renaming columns (using `.rename()` or assignment).
* **Summary Statistics:** Calculating descriptive stats on the columns (e.g., `.describe()`).

### üîç Notebook 02: Selection, Filtering, and Sorting
The second notebook applies these fundamentals to real-world data manipulation tasks:

* **Jupyter/Colab Features:** Helpful Jupyter features (e.g., magics, shortcuts).
* **Data Import:** Importing a CSV dataset into a DataFrame.
* **Advanced Selection:** Selecting multiple columns (passing a list of column names).
* **Boolean Logic:** Counting `True` values in a Boolean series.
* **Filtering:** Filtering rows with **Boolean masks**.
* **Complex Conditions:** Using **AND** (`&`) and **OR** (`|`) operators to create complex filtering criteria.
* **Copying Data:** Understanding why and when to use `.copy()`.
* **Sorting:** Sorting a DataFrame by one or more columns (using `.sort_values()`).
