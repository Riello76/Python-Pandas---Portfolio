## ‚ú® Key Concepts Covered

### üõ† Notebook 01: Creation, Properties, and Basic Manipulation

- **DataFrame Creation:** Create pandas DataFrames from lists, dictionaries, and other Python collections.  
- **Inspection:** Check DataFrame properties and information (`.shape`, `.dtypes`, `.info()`).  
- **Column Operations:** Select and manipulate individual columns.  
- **Modification:** Add new columns to a DataFrame.  
- **Renaming:** Rename columns using `.rename()` or assignment.  
- **Summary Statistics:** Calculate descriptive statistics using `.describe()`.  

### üîç Notebook 02: Selection, Filtering, and Sorting

- **Jupyter/Colab Features:** Magics and shortcuts for efficient workflow.  
- **Data Import:** Load CSV datasets into DataFrames.  
- **Advanced Selection:** Select multiple columns by passing a list.  
- **Boolean Logic:** Count `True` values in a Boolean series.  
- **Filtering:** Filter rows using Boolean masks.  
- **Complex Conditions:** Combine conditions using **AND** (`&`) and **OR** (`|`).  
- **Copying Data:** Understand when to use `.copy()`.  
- **Sorting:** Sort a DataFrame by one or multiple columns using `.sort_values()`.
